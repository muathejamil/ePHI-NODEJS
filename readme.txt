This is a light weight CRUD project for ePHI
the file ephi has a JSON object which you could use in this project to do all the operations.
project is running on port 5000
GET http://localhost:5000/ephis/   //to get all ephis
POST http://localhost:5000/ephis/   //to create a new ephi
{ephi.json}
GET http://localhost:5000/ephis/{id}  //to get ePHI by id
PATCH http://localhost:5000/ephis/{id}  // for update records
{ephi.json}
